About Dataset

This dataset has been created by merging the census 2011 of Indian Cities with Population more than 1 Lac and City wise number of Graduates from the Census 2011. The Census is one of the biggest workout conducted by the Government of India to understand the demographics of the country. It is conducted every 10 years. The Britishers initiated it during the colonial times. It helps in a big way to come up with social and economic programs in the country. It also helps to identify the problems in the country. We can use this dataset to create a visualization to see where the future cities of India stand today. More information can be found

http://www.census2011.co.in

About the Visualization

This visualization project is following the Interactive Slide Show narrative visualization structure, where user can navigate the slides by either clicking on the circle shown on right hand side or by using UP/DOWN arrow keys. Here every page has a title page followed by charts showing the details. All the charts provided are interactive, user can drill down in the charts using mouse events like mouse hover or in some cases mouse click.

The title scenes provides the author driven content. And the charts provides the reader driven content, which allows users to interact with data and draw their own conclusions.

All the chart are built using d3.js and HTML5

Scenes

Here the scenes are presented using slides. Eash slide has a provides the information related to the dataset. Slides are created using a particular templates where title has been followed by chart. The Title slide has different background color than content slide. User can navigate the scenes by scrolling the mouse in up and down direction.

Annotations

The annotation is represented by the legends and tooltip, which provides the more information about the data.

Triggers

The mouse movement and mouse click events act as a trigger to faciliate the trigger on the charts. By moving the mouse over the charts tooltip appears and disappears. Also, on clicking on the GeoMap of India other charts are updated with the detailed information, which helps in drilling down the information on the visualization presentation.

Parameters

In the Analysis page, when user clicks on the GeoMap, the parameters has been passed from one chart to other chart to get the relevant details. Also, state name has been passed in all the javascripts as a parameter to fetch the information related to particular state and shows that information on tooltips.
